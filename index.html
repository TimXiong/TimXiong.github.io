Markdown Editing

简述

提高工作效率，提供语法的高亮显示，支持快捷键插入 Markdown 标记，自带 code snippet。更详细的介绍请参考官方文档

安装

通过 Ctrl+Shift+p 打开命令面板，找到 Package Control: install Pakage ，然后搜索 markdown editing 点击安装，重启 Sublime Text，如果在 Preferences -> Package Settings 里面看到了 Markdown Editing 就说明安装成功了
