Markdown 编辑利器 Visual Studio Code

发表于 2016-12-15 | 分类于 Visual Studio Code ， Code Editor | | 阅读次数

Visual Studio Code 是微软推出的免费的轻量级编辑器，鉴于 VS IDE届老大的地位，VS Code 完全有潜力成为新一代编辑神器。
VS Code 原生支持 Markdown，安装后就可以直接使用，这里介绍几个常用的功能。

原生功能

预览

  • 在VS Code里面直接预览效果 Ctrl+Shift+V
  • 在VS Code里面分屏预览效果，并且可以实时修改 Ctrl+K then V

Snippets

  • 内置Snippets Ctrl+Space (Trigger Suggest) 比如输入 link ，然后选择 snippet，会自动插入 [text](http://link)
  • 自定的Snippets, 可以参考官方指南

插件

Markdown Theme Kit

可以在VS Code中使用和 Markdown Editing (Sublime Text 插件) 一样的皮肤。

Markdolint

非常值得推荐的插件，用于检查 Markdown 的格式规范，并提出建议，建议和规则来源于markdowlint for Ruby.
这里只列举几个常用的功能

  • 显示所有的错误和提示信息 Ctrl+Shift+M
  • 点击最左边的黄色提示会通过浏览器转到 github 上，查看更详细的建议

关于VS Code 对于 Markdown 更多的内容，可以参考其 官方网站
